# Marketing Campaign Analysis 
 
## Project Overview 
This project analyzes customer data to understand factors influencing customer acquisition and campaign performance. 
 
## Dataset 
The dataset (`marketing_data.csv`) contains information on demographics, product spending, sales channels, and campaign responses.
